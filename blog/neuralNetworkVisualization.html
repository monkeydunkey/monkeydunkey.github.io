Neural Network - Universal Approximator

Visualizing the output of a single layer Neural Network

The universal approximation theorem states that a neural network with a single hidden layer with finite number of neurons can approximate any continuous function. To check this theorem I built a visualization using D3.js, where in one can add neurons to a hidden layer, adjust the weight and bias term and try to approximate any continous function
